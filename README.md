# Smart Learning



Dillinger is a e-learning platform made in Javascript technology. Technology used in development:

  - NodeJS
  - MongoDB
  - Bootsrap
  - Javascript
  - HTML/CSS

# How to run!

  - Git clone a repository
  - Start Mongo service
  ```sh
$ mongo
```
  - start installation of packages with 
  ```sh
$ npm install
```
  - and start aplication 
  ```sh
$ npm start
```
  - Enjoy!
